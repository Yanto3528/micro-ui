# useTheme

useTheme is basically a hooks to get the current theme object passed to ThemeProvider

## Example

```jsx
import { Button } from '@aia-digital/component-library'
import { useTheme } from '@aia-digital/component-library/hooks'

function TestComponent() {
  // you can get the theme object passed to ThemeProvider
  const theme = useTheme()

  return (
    <Button customStyle={{ 'background-color': theme.colors.secondary }}>
      Click me
    </Button>
  )
}
```
